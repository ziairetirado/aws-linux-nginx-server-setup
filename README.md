# AWS EC2 Linux Server Deployment

This project demonstrates deploying and configuring a Linux server on AWS EC2.

## What I did:
- Launched EC2 instance (Amazon Linux 2023)
- Configured security groups
- SSH access
- Installed and configured NGINX
- Verified service running

## Commands Used:

sudo yum update -y
sudo yum install nginx -y
sudo systemctl start nginx
sudo systemctl enable nginx
